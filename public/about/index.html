About

This site has details about the sagepoint-node, which is a node validator for the Crypto.com testnet running in Connecticut. Statistics about the performance of the node can be found on the Statistics link. The node is running on a MacBook i7 with 8Gb of memory running Ubuntu 20.024. More details about he build are available in the Guide link. You can view the sagepoint-node on the Crypto.com Explorer site as well.
